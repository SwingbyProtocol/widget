Swingby's swap widget.

## Getting started

The easiest way to integrate the widget into your app is
[our dedicated SDK](https://github.com/SwingbyProtocol/widget-sdk).

### Minimum sizes

⚠️ Note that both the widget uses `rem` units.

|        |           |
| ------ | --------- |
| Banner | 320x76px  |
| Small  | 320x375px |
| Big    | 320x510px |
